# Инструкция по установке и запуску

## Шаг 1: Установите Node.js

Если Node.js еще не установлен, скачайте и установите его с официального сайта:
**https://nodejs.org/**

Рекомендуется версия LTS (Long Term Support).

После установки перезапустите терминал.

## Шаг 2: Проверьте установку

Откройте новый терминал и выполните:
```bash
node --version
npm --version
```

Если команды работают, Node.js установлен правильно.

## Шаг 3: Установите зависимости проекта

### Вариант 1: Используйте готовый скрипт
Дважды кликните на файл `install.bat` или выполните в терминале:
```bash
install.bat
```

### Вариант 2: Вручную
Откройте терминал в папке проекта и выполните:
```bash
npm install
```

## Шаг 4: Запустите проект

### Вариант 1: Используйте готовый скрипт
Дважды кликните на файл `start.bat` или выполните в терминале:
```bash
start.bat
```

### Вариант 2: Вручную
```bash
npm run dev
```

## Шаг 5: Откройте браузер

Перейдите по адресу: **http://localhost:3000**

---

## Если возникли проблемы

### Проблема: "npm не является внутренней или внешней командой"
**Решение**: Node.js не установлен или не добавлен в PATH. Установите Node.js и перезапустите терминал.

### Проблема: Ошибки при установке зависимостей
**Решение**: 
1. Убедитесь, что у вас установлена последняя версия Node.js
2. Попробуйте удалить папку `node_modules` (если есть) и выполнить `npm install` снова
3. Проверьте подключение к интернету

### Проблема: Порт 3000 уже занят
**Решение**: 
1. Закройте другое приложение, использующее порт 3000
2. Или измените порт в `package.json`: `"dev": "next dev -p 3001"`
