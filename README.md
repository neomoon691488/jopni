# Школьная Социальная Сеть

Социальная сеть для общения школьников, созданная на Next.js 14.

## Возможности

- ✅ Регистрация и авторизация пользователей с модерацией
- ✅ Создание и просмотр постов с изображениями
- ✅ Лайки и комментарии к постам
- ✅ Редактирование и удаление постов
- ✅ Профиль пользователя с редактированием
- ✅ Просмотр профилей других пользователей
- ✅ Поиск пользователей
- ✅ Система друзей
- ✅ Личные сообщения (чат в реальном времени)
- ✅ Панель администратора для модерации
- ✅ Загрузка изображений (аватары и посты)
- ✅ Красивый современный UI

## Установка и запуск локально

### 1. Установите зависимости

```bash
npm install
```

### 2. (Опционально) Создайте файл `.env.local`

```
JWT_SECRET=your-secret-key-change-in-production
```

Если файл не создан, будет использован дефолтный ключ для разработки.

### 3. Запустите сервер разработки

```bash
npm run dev
```

### 4. Откройте браузер

Перейдите по адресу: [http://localhost:3000](http://localhost:3000)

## Деплой в интернет

См. файл [DEPLOY.md](./DEPLOY.md) для подробных инструкций по деплою.

**Быстрый деплой на Vercel:**
1. Зарегистрируйтесь на https://vercel.com
2. Подключите GitHub репозиторий
3. Нажмите "Deploy"
4. Добавьте переменную окружения `JWT_SECRET`
5. Готово! Сайт доступен всем в интернете

## Использование

1. **Регистрация**: Первый пользователь автоматически становится администратором
2. **Модерация**: Администратор одобряет новых пользователей в `/admin`
3. **Создание постов**: На главной странице введите текст и добавьте фото
4. **Взаимодействие**: Лайкайте и комментируйте посты
5. **Друзья**: Добавляйте друзей через поиск или профили
6. **Сообщения**: Общайтесь с друзьями в личных чатах
7. **Профиль**: Редактируйте свой профиль, добавляйте аватар

## Технологии

- Next.js 14 (App Router)
- TypeScript
- React 18
- JWT для аутентификации
- bcryptjs для хеширования паролей
- JSON файлы для хранения данных (можно заменить на MongoDB/PostgreSQL)

## Структура проекта

- `app/` - страницы и API routes
- `components/` - React компоненты
- `lib/` - утилиты (база данных, аутентификация)
- `data/` - JSON файлы с данными (создается автоматически)
- `public/uploads/` - загруженные изображения

## Скрипты

- `npm run dev` - запуск сервера разработки
- `npm run build` - сборка для продакшена
- `npm start` - запуск продакшен сервера
- `npm run lint` - проверка кода линтером

## Важные замечания

⚠️ **Для продакшена рекомендуется:**
- Заменить файловую БД на MongoDB или PostgreSQL
- Настроить резервное копирование данных
- Использовать CDN для изображений
- Настроить мониторинг и логирование
